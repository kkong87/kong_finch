# Kevin Kong Finch App
Dockerized Flask App using [Finch APIs](https://developer.tryfinch.com/api-reference/organization/company) to pull in Company and Employee Information


## Usage

    docker compose up -d

.env must be updated to include your correct `client_id` , `client_secrect` , `customer_id` , `customer_name`  

## Access Browser

[http://localhost:5000](http://localhost:5000)
